Footer Menu Management -------------------------------------- -- Displays list of Footer Menus used for Front Page and other pages. -- Click on Column Headings to sort the list on that column. Click again to reverse the order. -- It contains the Link Text, Link URL, if the link is Offer Page for a category(channel) and the sorting order of the menu in Front Page and Offer Page. -- You will be able to Add/Edit/Delete this information according to your rights. -- While adding Footer Menu, you can specify Link URL or select the Offer Category, if the link URL is Offer Page. -- Check checkboxes for Front Page Display and Offer Page Display. Sort Footer Menu -------------------------- -- You can specify the order of the Footer Menu in Front Page and Offer Page. -- Leave the Sort Order Blank if you don't want the menu item to be displayed. -- You can display a Menu Item in FrontPageDisplay/OfferPageDisplay by specifying Sort Order. -- You can remove Menu Items from FrontPage/OfferPage by checking the checkbox against it and clicking Save/SaveNew. *********************************************************************************************************
