You Won Administration ----------------------------------- eMail Contents ---------------------- -- Displays the eMails used in You Won system. -- Click on Column Headings to sort the list on that column. Click again to reverse the order. -- [EMAIL_LINK] in the Message Body, will be replaced by user eMail address, when eMail content will be used to send an eMail. Set PopUp Delay ------------------------- -- Sets PopUp Delay (in Seconds), for YouWon Page 3. Manage index.php Text ---------------------------------- -- To Change the text displayed in YouWon index page. Edit index.php Headline ---------------------------------- -- To change the heading of YouWon index page. Edit PopUp Text ------------------------ -- To change the Text in PopUp window of the Prize to download. You Won Reporting ---------------------------- -- Report displays Datewise..., - No. of YouWon form submission - No. of users clicked on the link sent in the responding eMail. - Percentage of Users Clicked on the link. -- You can select the date range and Contact Form for the report. -- By Default it displys current month report for all Contact Forms. *********************************************************************************************************
