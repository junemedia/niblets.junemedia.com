Offer Management --------------------------- -- Displays list of Offers. -- Click on Column Headings to sort the list on that column. Click again to reverse the order. -- You will be able to Add/Edit/Delete this information according to your rights. -- While adding/editing, Check the checkboxes of Categorries, in which offer to be displayed. -- You can filter the list with Option of Exact Match. -- You can also filter the list with exclude. -- Filter will be applied to the field selected with Search In. If that field contains/exactly matches/excludes the filter text you entered. Offer Code Formula ---------------------------- -- Offer Code will be generated as per following formula : First 3 Characters are Company Code Next 1 Character is Placeholder - x Next 1 Character is Department code, which is 'e' for Editorial Next 6 Characters represents the Date(mmddyy) on which record created for the offer. Next 3 Characters are Sequence no. of a Company's offers. Reports ------------ Offer Redirects Report --------------------------------- -- Displays Report for the selected date range with two options. - History Report Displays report for the selected date range upto yesterday only. - Today's Report Displays report for TODAY ONLY, even if you select another date range. -- By default it displays History report of yesterday. -- By default it reports for all the companies. You can select one company to view report for that company only. -- Click on Column Headings to sort the list on that column. Click again to reverse the order. -- You can view the report for a specific company or filter by Offer Code. -- You can filter report on OfferCode with options of Starts With or Exact Match. Offers Expiring Report --------------------------------- -- Displays report of the offers which are expiring in next 30 days. -- Click on Column Headings to sort the list on that column. Click again to reverse the order. Frame Management ----------------------------- -- Manage common Top, Left, Right and Bottom frame contents for Offer page. -- The frame with specified content will be displayed if 'Display Frame' option is selected in Offer entry. *********************************************************************************************************
