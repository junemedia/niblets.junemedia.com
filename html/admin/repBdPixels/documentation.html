BD Pixel Tracking -------------------------- -- Displays Pixel Report for the selected date range with two options. - History Report Displays report for the selected date range upto yesterday only. - Today's Report Displays report for TODAY ONLY, even if you select another date range. -- By default it displays History report of yesterday. -- By default it reports for all the companies. -- You can select one company to view report for that company only. -- Click on Column Headings to sort the list on that column. Click again to reverse the order. -- You can filter report on SourceCode with options of Starts With or Exact Match. *********************************************************************************************************
